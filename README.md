#  ⏰ Issue-Tracker Project

## ✅ 프로젝트 소개
- 진행하고 있는 프로젝트의 이슈를 등록하고 관리할 수 있는 협업툴입니다
- 이슈를 등록하고, 등록된 이슈에 대해 댓글로 다른 유저들과 토론할 수 있습니다
- 이슈에 마일스톤, 라벨, 담당자 정보를 할당하여 보다 효율적으로 이슈를 관리할 수 있습니다
- 이슈의 부가정보(마일스톤, 라벨 담당자 등)에 따라 필터링하여 이슈를 검색할 수 있습니다
